EECS484: Database Management Systems, Fall 2018

Course Overview

12/18/2018

EECS 484 is an undergraduate level database course, taught by Prof. H.V. Jagadish. The course consists of two parts. The first part could be concluded as "how to use a database management system", while the second part could be "what is the internal design of database management system". Roughly speaking, the speed of the first part is too slow. We spent half of the semester learning relational calculus/algebra, sql syntax and normalization. The second part is good, and I learn a lot about the internal design. Topics include index and storage system of database management system, query processing, query optimization, transactions, schedules, and fault tolerance. I still think I need to learn more basic knowledge in computer systems. EECS 484 is a moderate workload but useful course under that scope. Considering learning CMU's CSAPP.
