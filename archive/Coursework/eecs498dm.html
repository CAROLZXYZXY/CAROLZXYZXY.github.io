EECS498:Data Mining, Fall 2018

Course Overview

12/18/2018

EECS 498-data mining is an undergraduate level data mining, taught by Prof. Danai Koutra. Topics include high dimensional data processing (LSH, clustering, dimensionality reduction), predictive data mining / machine learning (SVM, Decision Tree, Perceptron KNN), graph mining (PageRank, HIIT, Network Anlysis, Community Detection), and application of data mining (Association Rule, duplicate document detection). Compared with classical courses such as database and operating systems, those are the 21st century topics under rapid development. I learn a lot from this course. As for the project, using hadoop is a fantastic choice for us to know about this tool. (contributed partly by Mike). As for some minor problems, course management is not good, TAs always write bug on AG. Speed is low sometimes, because the lectures are 3-hour long. Every time I feel I learn nothing in the last hour. As for the final exam, I really want to see the answers, because peronalized pagerank is too magic!
One thing I appreciate a lot is that Danai provided us the opportunity of meeting her alone. I chatted with her a lot.
I'm thinking of TAing this course next fall.
